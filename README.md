# MessageThread

## 🛠️ 프로젝트 빌드 설정 (Build Configurations)

특정 환경(노트북 등)에서의 컴파일 에러 방지 및 원활한 빌드를 위해 Visual Studio 프로젝트 속성을 아래와 같이 변경해야 합니다.

### 1. SDL(Security Development Lifecycle) 검사 비활성화

일부 개발 환경에서 보안 검사(SDL)로 인해 빌드가 실패하거나 불필요한 경고/에러가 발생할 수 있어 해당 옵션을 비활성화합니다.

* **설정 대상**: C/C++ 컴파일러 보안 옵션
* **설정 방법**:
  1. Visual Studio 상단 메뉴에서 `프로젝트(P)` -> `[프로젝트 이름] 속성(P)` 선택 (또는 `Alt + Enter`)
  2. 좌측 트리 메뉴에서 **`구성 속성`** -> **`C/C++`** -> **`일반`** 이동
  3. 우측 항목 중 **`SDL 검사`**를 찾아 **`아니요(/sdl-)`**로 변경
  4. 우측 하단의 `적용(A)` 및 `확인` 버튼을 눌러 저장


>  **참고**: 해당 설정은 `활성(Debug)` 및 `활성(x64)`을 포함한 모든 빌드 구성(Configuration)에 공통으로 확인해 주는 것이 좋습니다.
